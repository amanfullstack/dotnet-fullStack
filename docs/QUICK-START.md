## 🚀 Quick Start: Deploy in 5 Minutes

### Option 1: GitHub Pages (FREE) ⭐ Recommended

```bash
cd DocumentationWebsite

# Step 1: Initialize Git
git init
git add .
git commit -m "Initial commit: .NET Learning Hub"

# Step 2: Push to GitHub
# (Follow DEPLOY.md for detailed steps)
git remote add origin https://github.com/YOUR_USERNAME/dotnet-learning-hub.git
git push -u origin main

# Step 3: Enable GitHub Pages
# In GitHub repo → Settings → Pages → Choose main branch
# Wait 1-2 minutes...

# Step 4: Visit your site!
# https://YOUR_USERNAME.github.io/dotnet-learning-hub
```

### Option 2: Test Locally First

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Then open: http://localhost:8000
```

See **DEPLOY.md** for detailed deployment instructions!

## 🛠️ How to Customize

### Change Colors

Edit `css/styles.css` - look for `:root { ... }`:
```css
--primary-color: #0d47a1;      /* Your brand color */
--secondary-color: #1976d2;    /* Accent color */
--success-color: #10b981;      /* Green for success */
```

### Change Content

All content is in `index.html`:
- Phase descriptions
- Interview Q&A
- Resource links
- Roadmap text

### Add Images

Create `images/` folder and add:
```html
<img src="images/your-image.png" alt="Description">
```

See **EXPAND.md** for detailed customization guide!
