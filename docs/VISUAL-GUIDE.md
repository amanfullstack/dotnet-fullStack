## **If It's Not Working**

### **Button doesn't show:**
1. Scroll page (make sure you can see hero section)
2. Refresh browser (Ctrl+R)
3. Try different browser

### **Docs page appears blank:**
1. Refresh page (Ctrl+R)
2. Clear browser cache
3. Try different browser

### **Text is overlapping:**
1. Try zooming in/out (Ctrl + or Ctrl -)
2. Resize window
3. Try different browser

### **Mobile view is bad:**
1. Try rotating device
2. Try different mobile browser
3. Zoom out if needed
