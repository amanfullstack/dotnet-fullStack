🖥️ ASP.NET MVC - Server-Side Web Application

A modern ASP.NET MVC 8 web application demonstrating server-side rendering, Razor views, and RESTful integration with ProductCatalogAPI.

Complete .NET 8
💡

What is ASP.NET MVC?

MVC (Model-View-Controller) separates your application into three layers. The server renders HTML and sends complete web pages to the browser. Perfect for traditional web applications with rich user experiences.

Features

  • ✅ Product List with Search & Filter
  • ✅ Create, Edit, Delete Products
  • ✅ Detail View for Individual Products
  • ✅ Dark/Light Theme Toggle
  • ✅ Responsive Bootstrap 5 Design
  • ✅ Dependency Injection & Service Layer
  • ✅ Error Handling & Validation

Tech Stack

Component Technology
Framework ASP.NET MVC 8 (.NET 8)
Language C# 12
UI Framework Bootstrap 5
Templating Razor Views
Backend ProductCatalogAPI REST

Getting Started

# Clone and navigate
cd ProductCatalogMVC-ASP.NET-MVC

# Run the application
dotnet run

# Open browser to https://localhost:5001

Project Structure

  • Controllers/ - Handle HTTP requests (ProductsController)
  • Models/ - Data models (Product, ProductViewModel)
  • Views/ - Razor templates (HTML + C#)
  • Services/ - Business logic (ProductService)
  • wwwroot/ - Static files (CSS, JS)
