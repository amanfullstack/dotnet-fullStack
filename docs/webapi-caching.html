    ⚡ Caching Strategies

    Five caching layers implemented in ProductCatalogAPI. From in-memory to Redis — reduce DB load by 99% and speed up responses 100x.

    Intermediate Web API
    ⚡

    What we achieved

    100 requests without cache = 15 seconds (100 × 150ms DB query). 100 requests with cache = 150ms (1 DB query + 99 cache hits). That's a 100x speedup and a 99% reduction in database load.

    1. In-Memory Cache (IMemoryCache)

    The simplest caching layer. Data is stored in the web server's process memory. Fast, zero setup, but lost on app restart and not shared between servers.

    Program.cs — Registration 
    builder.Services.AddMemoryCache(); // Registers IMemoryCache as Singleton
    Services/ProductService.cs — Cache-Aside Pattern 
    private readonly IMemoryCache _cache;
private const string ALL_PRODUCTS_KEY = "all_products";
private static readonly TimeSpan CacheDuration = TimeSpan.FromMinutes(5);

public async Task<IEnumerable<Product>> GetAllAsync()
{
    // 1. Check cache first
    if (_cache.TryGetValue(ALL_PRODUCTS_KEY, out IEnumerable<Product>? cached))
    {
        Console.WriteLine("✓ Cache HIT: all_products");
        return cached!;
    }

    // 2. Cache MISS — query database
    Console.WriteLine("✗ Cache MISS: querying database...");
    var products = await _context.Products
        .AsNoTracking()
        .OrderByDescending(p => p.CreatedAt)
        .ToListAsync();

    // 3. Store in cache with expiration
    var options = new MemoryCacheEntryOptions
    {
        AbsoluteExpirationRelativeToNow = CacheDuration,
        SlidingExpiration = TimeSpan.FromMinutes(2), // Reset TTL on access
        Priority = CacheItemPriority.Normal
    };
    _cache.Set(ALL_PRODUCTS_KEY, products, options);

    return products;
}

// Cache invalidation — call on Create, Update, Delete
private void InvalidateCache()
{
    _cache.Remove(ALL_PRODUCTS_KEY);
    Console.WriteLine("🗑️ Cache invalidated: all_products");
}

    2. HTTP Response Caching

    Client-side and proxy caching via standard HTTP headers. The browser or CDN caches the response — zero server load on subsequent requests.

    Response Cache Attributes on Controller 
    // Cache GET responses for 5 minutes
[HttpGet]
[ResponseCache(Duration = 300, Location = ResponseCacheLocation.Any)]
public async Task<ActionResult<IEnumerable<Product>>> GetAll() { ... }

// Never cache mutating operations
[HttpPost]
[ResponseCache(NoStore = true, Location = ResponseCacheLocation.None)]
public async Task<ActionResult<Product>> Create([FromBody] CreateProductRequest req) { ... }

// Vary by route parameter (cache per ID)
[HttpGet("{id:int}")]
[ResponseCache(Duration = 300, VaryByHeader = "Accept")]
public async Task<ActionResult<Product>> GetById(int id) { ... }

    3. Response Caching Middleware

    Server-side response caching. Stores full HTTP responses in memory. Requires [ResponseCache] on actions to activate.

    Program.cs 
    // Register service
builder.Services.AddResponseCaching();

// Add to pipeline (before app.MapControllers())
app.UseResponseCaching();

    4. Distributed Cache (Redis)

    Cache shared across multiple server instances. Data survives app restarts. Essential for production scale-out scenarios.

    NuGet + Program.cs 
    // Install: dotnet add package Microsoft.Extensions.Caching.StackExchangeRedis

builder.Services.AddStackExchangeRedisCache(options =>
{
    options.Configuration = builder.Configuration.GetConnectionString("Redis");
    // "Redis": "localhost:6379" in appsettings.json
    options.InstanceName = "ProductCatalog:";
});
    Using IDistributedCache 
    private readonly IDistributedCache _cache;

public async Task<Product?> GetByIdAsync(int id)
{
    var key       = $"product:{id}";
    var cachedJson = await _cache.GetStringAsync(key);

    if (cachedJson != null)
        return JsonSerializer.Deserialize<Product>(cachedJson);

    var product = await _context.Products.FindAsync(id);
    if (product == null) return null;

    var options = new DistributedCacheEntryOptions
    {
        AbsoluteExpirationRelativeToNow = TimeSpan.FromMinutes(10)
    };
    await _cache.SetStringAsync(key, JsonSerializer.Serialize(product), options);
    return product;
}
