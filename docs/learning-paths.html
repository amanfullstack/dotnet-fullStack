    📚 Recommended Topic Order

    If you're starting from scratch, follow this order — each topic builds on the previous:

    • 1
      Collections in C# Beginner ~1 hour

      Foundation — List, Dictionary, HashSet, IEnumerable vs IQueryable. You'll use these everywhere.

    • 2
      LINQ & Filtering Intermediate ~2 hours

      Query data from any collection or database. Deferred execution is key to EF Core performance.

    • 3
      Entity Framework Core Intermediate ~3 hours

      The primary data access method. Code-First, CRUD, migrations, change tracking, soft delete, relationships.

    • 4
      ADO.NET Intermediate ~2 hours

      Raw SQL for performance-critical paths. Parameterized queries (security), transactions, connection pooling.

    • 5
      Performance Comparison Advanced ~1.5 hours

      Understand the real numbers. Caching impact, N+1 problem, decision matrix for choosing technologies.

    • 6
      Interview Q&A Interview ~2 hours

      35+ questions covering everything above. Read only after studying all topics — test yourself first.

    💡 Study Tips

    ✍️

    Type the code examples yourself

    Don't copy-paste. Type every code example manually. Your fingers and brain learn differently — muscle memory helps recall in interviews and when coding under pressure.

    🗣️

    Explain it out loud

    After each topic, close your laptop and explain it in your own words as if teaching someone. If you can't explain it simply, you haven't understood it yet. This is how you prepare for verbal interview questions.

    🔍

    Use search (Ctrl+K) during review

    This site has search built in. During review sessions, type a concept you're unsure about to find it quickly across all pages.

    🧠

    Understand WHY, not just WHAT

    Don't memorize code. Understand why each pattern exists. Why use IMemoryCache? Why does middleware order matter? Why use DTOs? Interviewers always follow with "why" — rote memorization won't get you through.

    🔗

    Connect to the real project

    Every topic in this site is implemented in ProductCatalogAPI. After reading a topic, find the relevant code in the project. Seeing patterns in real production code cements understanding better than any tutorial.

    📱

    Access from anywhere

    This site is hosted on GitHub Pages — it works on your phone, tablet, and any browser. Bookmark it and review during commutes, lunch, or any spare time. Consistent review beats marathon cramming.
